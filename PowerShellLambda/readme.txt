This is a scaffold for implementing a AWS PowerShell Lambda function/ application with Serverless Application Model (SAM).

The DeployLambda.ps1 script helps with **manually** deploy the AWS PowerShell Lambda application.
The prerequisites are:
.Net Core SDK / https://dotnet.microsoft.com/download
AWS CLI / https://docs.aws.amazon.com/cli/latest/userguide/install-windows.html
PowerShell Core 6 / https://github.com/PowerShell/PowerShell
PowerShell modules:
    awspowershell.netcore
    AWSLambdaPSCore
Pre-created S3 bucket in the target account.
